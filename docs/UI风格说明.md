# UI 风格说明

## 整体定位

> **现代科技风 · 年轻化 · 校园化 · 清爽高级**

本平台的 UI 设计面向大学生群体，追求简洁、现代、有科技感的视觉体验，与传统商城、管理后台风格彻底区分。

## 布局规范

### 基本原则

- **卡片式布局**：所有内容块使用卡片承载，卡片之间留有足够间距
- **大留白**：页面四周、模块之间保持充足留白，避免拥挤感
- **栅格系统**：使用 Element Plus 的 `el-row` / `el-col`，常见 24 栅格
- 内容区最大宽度 `1200px`，居中显示

### 卡片样式

```css
.card {
  background: #FFFFFF;
  border-radius: 16px;
  padding: 24px;
  box-shadow: 0 4px 20px rgba(108, 92, 231, 0.08);
  transition: all 0.3s ease;
}

.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 30px rgba(108, 92, 231, 0.15);
}
```

## 动效规范

### 原则：克制、流畅、有意义

- **过渡时间**：`0.3s` 为主，大动效 `0.5s`
- **缓动函数**：`ease` 或 `cubic-bezier(0.4, 0, 0.2, 1)`
- **Hover 效果**：轻微上浮 + 阴影加深，不夸张
- **页面切换**：淡入淡出，不使用花哨的翻转/缩放
- **加载态**：使用骨架屏（Skeleton），不使用转圈圈
- **数据更新**：数字变化使用轻微缩放动画

### 禁止

- 不要使用弹跳、旋转、翻转等夸张动效
- 不要使用自动轮播横幅
- 不要使用闪烁的促销标签
- 不要使用跑马灯效果

## 各页面风格要点

### 登录/注册页

- 全屏渐变背景（蓝紫 → 青 → 深蓝）
- 居中玻璃拟态卡片
- 简洁表单，大圆角输入框
- 校徽/Logo 置于顶部

### 首页

- 顶部搜索栏（大圆角，带阴影）
- 分类标签横向滚动（胶囊形）
- 商品卡片瀑布流或网格，每行 3-4 列
- 卡片内容：商品图片、标题、价格（主色加粗）、匹配度标签、地点
- 底部导航栏（移动端）或顶部导航栏（桌面端）

### 商品详情页

- 左侧商品图片轮播（大圆角）
- 右侧商品信息卡片
- 匹配度分数环形图或进度条
- 匹配原因标签列表
- 卖家信息卡片（头像 + 昵称 + 认证标）
- 操作按钮：联系卖家（主色渐变）、收藏（描边）

### 匹配结果页

- 顶部：求购需求摘要卡片
- 下方：匹配商品列表，每个卡片显示匹配分数和匹配原因
- 匹配分数用彩色进度条直观展示
- 按匹配度降序排列

### 管理后台

- 侧边栏：深色背景（`#1A1A2E`），白色文字，选中项高亮为渐变
- 顶部栏：白色背景，轻微阴影
- 内容区：浅灰蓝背景
- 数据卡片：白色卡片，彩色图标，数字加粗
- 图表：使用 ECharts，配色与主色调一致

### 个人中心

- 顶部个人信息卡片（渐变背景）
- 下方功能入口图标网格
- 订单/收藏/发布使用 Tab 切换

## 图标规范

- 使用 Element Plus Icons 为主
- 可搭配 iconfont 自定义图标
- 图标颜色跟随主色调
- 图标大小统一：导航 20px，卡片 16px，按钮内 14px

---

## 设计禁忌

| 禁止 | 替代方案 |
|------|----------|
| 纯白色背景大面积使用 | 使用 `#F5F6FA` 浅灰蓝 |
| 表格堆叠展示数据 | 使用卡片列表 |
| 直角边框 | 使用 10-16px 圆角 |
| 高饱和度颜色 | 使用柔和、低饱和度配色 |
| 密集文字排版 | 增加行高至 1.6-1.8，增大留白 |
| 传统红色/橙色促销色 | 使用蓝紫/青色系 |
| 默认 Element Plus 蓝色主题 | 覆盖为主色调 `#6C5CE7` |
| 纯黑文字 | 使用 `#2D3436` 深灰 |
