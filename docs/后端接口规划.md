### 四、文件上传模块

#### 1. 上传图片

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/upload/image` | 上传图片（需登录，form-data） |

**请求参数**：`file`（multipart/form-data）

**响应示例**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "url": "/upload/2026/06/xxx.jpg"
  }
}
```

**限制**：
- 仅支持 jpg、jpeg、png、webp 格式
- 文件大小不超过 5MB
- 按日期分目录存储，文件名使用 UUID

**访问**：上传后可通过 `http://localhost:8080/upload/2026/06/xxx.jpg` 直接访问图片

## Token 使用方式

需要登录的接口在请求头携带 Token：

```
Authorization: Bearer <token>
```

## 权限说明

| 权限级别 | 可访问接口 |
|----------|-----------|
| 匿名访问 | `/api/health`、`/api/user/register`、`/api/user/login`、`/api/admin/login`、`/api/goods/list`、`/api/goods/detail/**`、`/upload/**` |
| 普通用户登录 | `/api/user/profile`、`/api/goods/add`、`/api/goods/update/**`、`/api/goods/delete/**`、`/api/goods/my`、`/api/demand/**`、`/api/favorite/**`、`/api/order/**`、`/api/match/**`、`/api/upload/image` |
| 管理员登录 | `/api/admin/**`（`/api/admin/login` 除外） |

**未登录返回 401**：
```json
{
  "code": 401,
  "message": "未登录或登录已过期",
  "data": null
}
```

**无权限返回 403**：
```json
{
  "code": 403,
  "message": "无权限",
  "data": null
}
```

---

## 统一响应格式

所有接口统一使用以下格式：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

状态码：
- 200：成功
- 400：参数错误
- 401：未登录或 Token 已过期
- 403：无权限
- 404：资源不存在
- 500：服务器内部错误
